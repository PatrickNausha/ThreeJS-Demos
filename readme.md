# Run locally

1. Run `npm install`
1. Run `npm i -g parcel`
    * This app was last confirmed building with parcel version 2.4.1.
    * TODO: Avoid using a globally installed build too.
1. Run `npm run dev`
1. Open a browser to http://localhost:1234/

# Deploy

1. Run `npm run build`
1. Run `npm run deploy`
